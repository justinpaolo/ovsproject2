# ovsproject2
The Online Voting System (OVS) is a web-based application that enables secure and efficient voting. It allows users to register, vote for candidates, and view results, while administrators manage the entire election process.

User Side:
User registration and login
Google login integration
View candidates
Cast vote (one vote per user)
View election results
Profile management

Admin Side:
Admin authentication
Candidate management (Add, Edit, Delete, Approve)
Voter management (View, Validate, Delete)
Start/Stop voting
Reset votes
View results and archives

Technologies Used:
Backend: PHP
Frontend: PHP, CSS,
Database: MySQL
Framework/UI: Bootstrap (SB Admin 2)
Dependency Manager: Composer

Installation Guide
1. Clone the Repository
git clone https://github.com/yourusername/online-voting-system.git
2. Move to XAMPP

Place the project folder inside:

htdocs/
3. Setup Database
Open phpMyAdmin
Create a database (e.g., ovs_db)
Import the SQL file from:
/database/ovs.sql
4. Run the System

Open your browser and go to:

http://localhost/ovsproject2
